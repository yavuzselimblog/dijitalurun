CSS

This part of the documentation will help you understand the CSS structure.

If you do not want to use SASS then you can use CSS inside src/assets/css/**.css and src/plugin/**.css. You can directly make changes in custom.css file which we have provided inside assets/css/custom.css. We suggest you do not make any changes in the app, bootstrap or other CSS files.
